# 🧠 Skills & Design Intelligence — Decor and Modules Website

## 🎯 Objective

Build a **visually immersive, scroll-driven static website** for an interior design company that feels **premium, modern, and non-generic**.

The design must prioritize:

* Visual storytelling
* Smooth animations
* Rich imagery
* Clean spacing
* High-end brand feel

Avoid template-like layouts.

---

## Core Design Philosophy

### 1. Scroll-Based Storytelling

* Each section should feel like a **scene transition**
* Use **full-screen sections (100vh)** where appropriate
* Smooth scroll behavior
* Sections should **flow naturally**, not feel stacked

---

### 2. Visual First Approach

* Images are the **primary content**
* Use:

  * Full-width banners
  * Image grids
  * Split image-text layouts
* Maintain consistent tone (luxury interiors)

---

### 3. Motion & Animation (Key Differentiator)

Use animations to enhance experience, not distract.

#### Required Animation Types:

* **Fade-in on scroll**
* **Slide-in (left/right/up)**
* **Zoom-in images on hover**
* **Parallax scrolling effects**
* **Text reveal animations**
* **Count-up animation for statistics**
* **Smooth hover transitions (scale, shadow, glow)**

#### Libraries (Preferred):

* Framer Motion (if React)
* AOS (Animate on Scroll)
* GSAP (for advanced effects)

---

## Section-Level Design Intelligence

### 🔹 Home (Hero Section)

* Full-screen hero with **image/video background**
* Overlay gradient for readability
* Animated captions (fade/slide rotation)
* Strong CTA: "Enquire Now"

#### Add:

* Scroll indicator (↓ arrow animation)
* Floating or inline **stats counter**:

  * Projects
  * Experience
  * Clients

---

### 🔹 Our Projects

* Use **masonry or asymmetric grid layout**
* Avoid simple equal grids

#### Effects:

* Image zoom on hover
* Overlay with project title
* Lightbox preview on click

---

### 🔹 Services

* Use **card-based layout with depth**

#### Effects:

* Hover lift (translateY + shadow)
* Icon animation (scale/rotate slightly)
* Smooth transitions

---

### 🔹 About

* Split layout (image + text)
* Add **subtle parallax effect** on image

---

### 🔹 Why Us

* Use **icon + text blocks**
* Animate each item on scroll (stagger effect)

---

### 🔹 Contact

* Clean form UI
* Input focus animations
* Floating labels (preferred)

#### Add:

* Floating WhatsApp button (bottom-right)
* Social icons with hover effects

---

## UI/UX Excellence

### Color Usage

* Minimal palette
* High contrast text
* Accent color used only for CTAs

---

### Typography

* Large, bold headings
* Generous line spacing
* Avoid dense text blocks

---

### Spacing Rules

* Use **large padding (80px–120px sections)**
* Avoid clutter
* Maintain breathing space

---

## Responsiveness Strategy

* Mobile-first adjustments:

  * Stack layouts vertically
  * Reduce animation intensity
* Ensure smooth scrolling on all devices

---

## Technical Implementation

### Layout

* Use Flexbox + CSS Grid
* Avoid rigid layouts

### Performance

* Optimize images
* Lazy load images
* Avoid heavy animation overload

---

## What to Avoid (IMPORTANT)

Do NOT:

* Use basic bootstrap-style layouts
* Use crowded sections
* Use too many colors
* Use static/plain sections without animation
* Make it look like a template

---

## Expected Outcome

The final website should feel like:

* A **luxury interior brand**
* Smooth and interactive
* Visually engaging from top to bottom
* Clearly better than generic template websites

---

## Bonus Enhancements (Optional but Powerful)

* Scroll-triggered background transitions
* Section separators using shapes or gradients
* Micro-interactions on buttons
* Cursor hover effects (advanced)
* Animated section transitions

---

## Final Instruction

Design with intention:
Every section must feel **alive, premium, and thoughtfully crafted**.

The user should feel:
👉 “This is a high-end interior design brand”
—not a basic website.
